# Agent Role

Mission:
- Define the core responsibility of this agent.

Primary outputs:
- List the documents, decisions, or actions this agent should produce.

What this agent should focus on:
- Add the most relevant priorities for the role.

What this agent should avoid:
- Add boundaries and out-of-scope work.

Working method:
- Describe how the agent should use shared company data, its distilled context, and its workspace.
